# 🏗️ Tugas Praktikum Bootstrap
Proyek tugas praktikum pembuatan website multi-tema menggunakan **Bootstrap** dengan tema layanan publik di **Kabupaten Bojonegoro, Jawa Timur**.

## 📋 Deskripsi

Website ini terdiri dari 4 halaman HTML statis yang mendemonstrasikan penggunaan berbagai komponen Bootstrap:

| Halaman | File | Deskripsi |
|---------|------|-----------|
| 🏠 **Landing Page** | `index.html` | Halaman utama dengan navigasi ke 3 tema |
| 🏥 **Rumah Sakit** | `rumah-sakit.html` | RSUD Dr. Sosodoro Djatikoesoemo Bojonegoro |
| 📚 **Perpustakaan** | `perpustakaan.html` | Perpustakaan Daerah Kab. Bojonegoro |
| ✏️ **Toko Alat Tulis** | `toko-alat-tulis.html` | Toko ATK Jaya Abadi Bojonegoro |

## 🛠️ Teknologi

- **HTML** — Struktur halaman
- **Bootstrap** — Framework CSS (CDN)
- **Bootstrap Icons** — Ikon (CDN)
- **Google Fonts (Poppins)** — Tipografi

## 🎨 Komponen Bootstrap yang Digunakan

### Layout
- Container, Grid System (`row`, `col-md-*`, `col-lg-*`, `col-sm-*`)

### Content
- Typography (`display-4`, `lead`, `fw-bold`, `fw-semibold`)
- Tables (`table-bordered`, `table-hover`, `table-striped`, `table-responsive`)

### Navigation
- Navbar (`navbar-expand-lg`, `navbar-dark`, collapse toggle)

### Forms
- Form Controls (`form-control`, `form-select`)
- Input Group (`input-group`, `input-group-text`)
- Checkbox (`form-check`) & Radio Button (`form-check-inline`)
- Textarea, Date Input, Number Input

### Components
- Cards (`card`, `card-body`, `card-header`)
- Buttons (`btn`, `btn-lg`, `btn-outline-*`, `rounded-pill`)
- Badges (`badge`, `rounded-pill`)
- Alerts (`alert-info`)
